signal framework - how to read this dashboard
green zone = accumulate / buy
yellow zone = maintain / watch
red zone = derisk / sell
composite score methodology: each indicator is scored 0-100 based on its zone position, then weighted by historical predictive power: sentiment & credit spreads (30%), rates & fed (25%), macro (25%), fiscal (15%), commodities (5%).

score thresholds: < 35 = green (accumulate), 35-55 = yellow (maintain), 55-70 = orange (reduce), > 70 = red (derisk).

how to read the threshold strip: each card shows a green/yellow/red bar. the colored dot shows where the indicator currently sits. if the dot is near a boundary, watch for a breakout - that is your early warning. click any card to see the "next level" text telling you exactly what data point would push it into the adjacent zone.
standalone dashboard - works out of the box. this file is fully self-contained with demo data. to add live data, you need a free FRED API key + a simple proxy server (browsers block direct API calls due to CORS). see the signal framework tab for setup instructions.
